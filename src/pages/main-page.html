구유 4기 트랙 소개

구름톤 유니브 4기 경인지부의 다양한 파트별
스터디에 대해서 소개해 드릴게요.

  • 애자일 스크럼과 칸반을 이해해 스프린트 계획 및 진행을 효율화해요.

  • 사용자 경험(UX) 기본 원칙을 학습해 PM 의사결정에 반영해요.

  • 웹 접근성(WCAG) 가이드라인을 숙지해 모두를 위한 제품을 설계해요.

  • Git 기반 버전 관리 흐름을 알아 개발팀과 원활히 협업해요.

  • CI/CD 파이프라인 개념을 이해해 배포 자동화 및 리스크를 줄여요.

  • 디자인 씽킹 프로세스를 익혀 문제 정의부터 솔루션 도출까지 구조화해요.

  • 사용자 조사(페르소나 · 여정 지도) 기법을 학습해 인사이트 기반 디자인을 만들어요.

  • 와이어프레이밍 · 프로토타이핑 툴(Figma·Sketch) 사용법을 숙지해 빠르게 시안 검증해요.

  • 반응형 그리드 시스템과 타이포그래피 원칙을 이해해 다양한 디바이스에 최적화해요.

  • 접근성(색 대비 · 키보드 내비게이션) 가이드라인을 적용해 모두가 사용할 수 있는 인터페이스를 설계해요.

  • 리액트로 CSS 라이브러리 적용 · 이벤트 핸들링 · 조건부 렌더링을 실습해요.

  • Styled-Components · 반응형 웹 · ESLint·Prettier 설정을 통해 코드 품질을 높여요.

  • JS 기초 · 비동기 처리 · JSX 컴포넌트로 카운터 · 모달 구현과 React 환경 설정을 해요.

  • HTML/CSS 마크업 · 웹접근성 · 시맨틱 태그 · Flex/Grid 반응형 레이아웃을 학습해요.

  • React Native로 스타일링 · 네비게이션 · 디바이스 API 연동을 실습해요.

  • SwiftUI로 뷰 레이아웃 · 상태 관리 · 데이터 바인딩을 배워요.

  • 크로스플랫폼 성능 최적화·디버깅·테스트를 적용해요.

  • Spring Boot 구조와 DI(의존성 주입) 개념을 익혀요.

  • REST API 설계 및 컨트롤러 작성법을 학습해요.

  • JPA/Hibernate로 데이터베이스 연동과 엔티티 매핑을 해요.

  • Spring Security로 인증 · 인가 기능을 구현해요.

  • 단위 · 통합 테스트와 CI/CD 파이프라인을 적용해요.
